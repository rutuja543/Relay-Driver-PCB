# 🔌 Relay Driver PCB

A relay driver circuit designed using **KiCad** to safely control AC/DC loads from low-power microcontrollers such as **Arduino**, **ESP32**, and **STM32**. The design uses an NPN transistor as a switching device and includes a flyback diode to protect the transistor from relay back EMF.

---

## 📌 Project Overview

This project demonstrates the design of a relay driver PCB that enables microcontrollers to switch high-current electrical loads safely. The PCB is compact, easy to assemble, and suitable for automation and embedded system applications.

---

## 🎯 Objectives

- Design a reliable relay driver PCB
- Interface low-voltage controllers with high-power loads
- Protect switching transistor using a flyback diode
- Learn PCB schematic and layout design using KiCad

- LED
- Current Limiting Resistor
- Screw Terminal Connectors
- Power Supply Connector

---

## 🔄 Working Principle

1. The microcontroller sends a HIGH signal.
2. The transistor turns ON.
3. Current flows through the relay coil.
4. The relay energizes and switches the external load.
5. The flyback diode suppresses reverse voltage generated by the relay coil during switching.
